# Pharmacy UI Prototype

This repository contains a presentation-focused frontend prototype for a Pharmacy Sales and Inventory System.

## Open in Browser

1. Open the `pharmacy-ui` folder.
2. Start a simple local server from the repository root, for example:
   - `python3 -m http.server 8000`
3. Visit `http://localhost:8000/pharmacy-ui/` in your browser.
4. Log in with any username/password and choose a role to enter the matching dashboard.

## Included Pages

- `pharmacy-ui/index.html` – login page
- `pharmacy-ui/admin.html` – admin dashboard
- `pharmacy-ui/pharmacist.html` – pharmacist dashboard
- `pharmacy-ui/staff.html` – staff dashboard
- `pharmacy-ui/products.html` – product management
- `pharmacy-ui/inventory.html` – inventory monitoring and reorder recommendations
- `pharmacy-ui/sales.html` – POS and receipt simulation
- `pharmacy-ui/reports.html` – report center

All data is powered by local mock JavaScript/JSON files only.
